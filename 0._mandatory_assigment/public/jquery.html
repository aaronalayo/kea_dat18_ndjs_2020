jQuery is a JavaScript Library

The jQuery library contains the following features:

HTML/DOM manipulation
CSS manipulation
HTML event methods
Effects and animations
AJAX
Utilities

How to use

In order to use jQuery properly a script must be pasted at the end of the body of every html file
by looking for it on google so it always be the lastest version.

script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js">

Then we need to create a file file.js that will contain all the jQuery features.

Then another script must follow the jQuery script which will have the path to this file.js

script src="./style.js">

Here is the jQuery used to modify this website
