Objects

Objects in JavaScript are collections of key/value pairs. The values can consist of properties 
and methods, and may contain all other JavaScript data types, such as strings, numbers, and Booleans.
All objects in JavaScript descend from the parent Object constructor.

Retrieve value from object by key

var myObj = {"message": "Hello, earthling! I bring peace."};

console.log(myObj.message);
//shows in the console
Hello, earthling! I bring peace.

console.log(myObj['message']);
//shows in the console
Hello, earthling! I bring peace.

Defining an object.

Create an object that has your name and age. 

var p = {
    "name" : "Aaron",
    "age": 37
};

console.log(p.name, p.age);
//shows in the console
Aaron, ALAYO

Adding a property

var stackOverflow;
var stackOverflow = {};
stackOverflow.isAllowed = true;

make a rule called isAllowed and let the value be true

console.log(stackOverflow.isAllowed);
//shows in the console
True

Removing a property

var thisSong = {"description": "The best song in the world."}

remove the property "description" and add a property called "about" that should say "Just a tribute."

delete thisSong.description;
thisSong["about"] = "Just a tribute";

console.log(thisSong.about);
//shows in the console
Just a tribute
